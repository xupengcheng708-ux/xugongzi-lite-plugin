# 抓整个号

给抖音主页分享链接 → 列出全部作品 → 学员按条件筛选 → 批量下载转写 → 归档到「抓账号」目录。

**核心原则**：先列后选。没选片之前不下载任何视频。

## 执行步骤

### 1. 解析参数

用户输入：`$ARGUMENTS`。提取：
- 主页 URL（`v.douyin.com/xxx` 短链 / `www.douyin.com/user/MS4w...`）
- 可选策略：`最新 10 条` / `近半年 top 20` / `点赞>5000` / 日期段等

### 2. 读配置

```bash
MODE=$(bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/lib/config.sh get mode)
ACCOUNT_DIR=$(bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/lib/config.sh get account_dir)
```

### 3. 列作品清单

**路线 A**（功能完整，有点赞/评论数）：
```bash
bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/mode_a/account.sh list "<主页URL>" "/tmp/xgz_list_$$.json"
```

**路线 B**（只有基础元数据：标题/日期/时长）：
```bash
bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/mode_b/account.sh list "<主页URL>" "/tmp/xgz_list_$$.json"
```

读 JSON 展示给学员（最多 30 条 + 总数）：
```
账号：<nickname> · 共 N 条作品
 1. 2025-XX-XX | <点赞数 仅路线A> | <标题前 40 字>
 2. ...
```

### 4. 按策略筛选

| 学员说 | 怎么筛 |
|---|---|
| 最新 N 条 | 按发布时间倒序取 N |
| 全部 | 无过滤（但要提醒：N×2 分钟转写成本） |
| 近半年 top N | 近 180 天 + 点赞降序（**仅 A 支持**；B 只能给"近半年全部"） |
| YYYY-MM 到 YYYY-MM | 日期段过滤 |
| 点赞>N | **仅 A 支持** |
| 这一条 + URL | 直接单条下载 |

路线 B 遇到不支持的策略 → 告诉学员「通用版拿不到点赞数，要按点赞筛请换路线 A，或者你手动挑」。

### 5. 展示候选清单 + 学员确认

```
按你的策略「最新 10 条」命中 10 条：
  1. 2025-04-20 | XXX赞 | 标题...
  2. ...
要下载这份吗？模式：保留视频 / 仅文案（默认仅文案，省 80% 磁盘）
```

**必须等学员明确"开始"/"下载"再往下走**。别自作主张。

### 6. 批量下载+转写（后台任务）

```bash
TASK_ID=$(bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/lib/task_log.sh create "抓整个号" "<主页URL> | N 条")
nohup bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/mode_<a|b>/account.sh download \
  "/tmp/xgz_list_$$.json" "$TASK_ID" "$ACCOUNT_DIR" \
  --ids "<逗号分隔的选中 aweme_id>" \
  [--audio-only | --keep-video] \
  > /dev/null 2>&1 &
```

告诉学员：
```
✅ 已启动批量下载：N 条视频
  任务 ID：<TASK_ID>
  预估时间：N × 2 分钟
  进度：/任务状态
  归档到：<account_dir>/<账号名>/
```

立即返回，不阻塞。

### 7. 后台脚本做什么

脚本自己串起来：
1. 按 ids 列表逐个下载（或只下音频）
2. whisper 转写每条
3. 生成 md 文件（frontmatter 含标题/日期/点赞数 等 + body 含文案）到 `<account_dir>/<账号名>/YYYY-MM-DD-<标题>.md`
4. 生成 `_索引.md`（按点赞降序的总览表）
5. 更新任务日志

## 注意

- 单次别超过 50 条（风控）
- 默认 `--audio-only` 省磁盘；学员要留视频自己传 `--keep-video`
- 抓完引导学员：「要跑对标拆解吗？这几条里爆款 top3 是 ...」
