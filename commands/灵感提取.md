# 灵感提取

接收单条视频 URL（抖音/B站/小红书/YouTube），下载 → 转写 → 生成带文案的 md 文件存到学员配置的「灵感池」目录。

## 执行步骤

### 1. 解析参数

用户输入：`$ARGUMENTS`

从输入里提取 URL。如果是抖音分享文案（"XX 为你推荐 https://v.douyin.com/XXX..."），用正则抽出 URL。

如果没给 URL，问学员："给我一个视频链接。"

### 2. 读配置

```bash
bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/lib/config.sh get mode
bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/lib/config.sh get inspiration_dir
```

如果 config 不存在 → 告诉学员先跑 `/xugongzi-init`。

### 3. 启动下载+转写（后台任务）

把任务扔后台，立刻返回任务 ID，学员可以用 `/任务状态` 查进度。

**路线 A**（config.mode = a）：
```bash
TASK_ID=$(bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/lib/task_log.sh create "灵感提取" "<URL>")
nohup bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/mode_a/inspiration.sh "$TASK_ID" "<URL>" > /dev/null 2>&1 &
```

**路线 B**（config.mode = b）：
```bash
TASK_ID=$(bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/lib/task_log.sh create "灵感提取" "<URL>")
nohup bash ${CLAUDE_PLUGIN_ROOT:-$HOME/.claude/plugins/xugongzi-lite}/scripts/mode_b/inspiration.sh "$TASK_ID" "<URL>" > /dev/null 2>&1 &
```

### 4. 告诉学员

```
✅ 任务已启动：<TASK_ID>
  视频：<URL>
  预估时间：30 秒～3 分钟（看视频长度）
  进度查看：/任务状态
  完成后会写到：<inspiration_dir>/YYYY-MM-DD-<视频标题>.md
```

### 5. 不要等任务完成

立刻回复学员，不要阻塞等转写结果。学员用 `/任务状态` 自己轮询。

## 产出的 md 长什么样

（脚本会自动生成这个结构）

```markdown
---
标题: <视频标题>
平台: <抖音/B站/小红书/YouTube>
链接: <URL>
提取时间: YYYY-MM-DD HH:MM
方式: subtitle / whisper
状态: 已提取
---

# <视频标题>

## 📄 原始文案

<带时间码的逐字稿>
```

学员拿到这个 md 后，可以手动或用 Claude 做进一步分析（钩子/结构/金句等）。本命令不跑分析，只抽文案。要带分析走 `/对标拆解`。
